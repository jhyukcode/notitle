  ## di (dependency injection : 의존성 주입)
    - 각 클래스간의 의존 관꼐를 [설정파일]을 통해 [컨테이너]가 자동으로 연결
      > 장점 : 코드 단순화 / 결합도 제거


      Class A{}
      A a1 = new A()

      생성 → 초기화 → 서비스 → 소멸
       ↑                       ↓
    -------------------------------
      ioc가 아닌 경우 [ 개발자 → (Class A)
                              → (Class B)
                               → (Class C) ] 컨테이너
      ioc인 경우      [ 개발자 ← (Class A)
                               ← (Class B)
                                ← (Class C)] 컨테이너

    1. bean

        -xml [운영]
