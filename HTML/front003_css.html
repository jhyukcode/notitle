1. font 속성

        1. 글자색 : color
        2. 글자사이즈 : font-size
        3. 기울임꼴 : font-style:italic / normal
        4. 굵기 : font-weight:bold / normal / light
        5. 글꼴 : font-family

2. 문단속성

        1. 행간 : line-height
        2. 정렬 : text-align
        3. 들여쓰기 : text-indent
        4. 밑줄 : text-decoration: underline / none / line-through
        5. 자간(글자간격) : letter-spacing
